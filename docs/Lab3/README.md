# Lab 3 — Build a Prompt-Only Agent: Weekly Stakeholder Update

**Goal:** Create a simple, linear **Copilot custom agent** with well-structured instructions, then run it in GitHub Copilot Chat inside VS Code.

This lab pairs with Module 2 (Section 1): **models, prompts, context, and the agent harness loop**.

## What you’ll build

A reusable “Weekly Stakeholder Update” agent that:

1. Accepts **as much or as little** input as you provide.
2. Asks smart follow-up questions to help you remember what happened last week.
3. Produces a polished weekly update in multiple formats.

## Files

- [Lab Guide](./LabGuide.md)

## Output (what you should end with)

- A workspace-scoped **custom agent** profile in `.github/agents/*.agent.md`
- A saved weekly update in markdown (created by your agent)
