# Prompt: Spec-driven

You are an AI coding agent. Use the Product Requirements Document at `docs/Lab1/PRD.md` as the single source of truth. Build the **Customer Feature Request Triage** web app per the PRD.

## Steps
1. Read the PRD and restate it as a checklist.
2. Propose a minimal architecture and libraries (briefly).
3. Implement backend + frontend to satisfy the PRD.
4. Add tests covering acceptance criteria (tag suggestion, filtering, dashboard counts).
5. Update README with setup, run, and test commands.
6. Self-check against acceptance criteria and note any gaps.

## Deliverables
- Source code
- README with commands
- Tests and/or test results
- Brief summary of what met/what’s pending
