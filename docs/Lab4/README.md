# Lab 4 — Competitive Intelligence Deck Builder (Work IQ + Skills + Skeptic)

**Goal:** Build a multi-agent Copilot workflow that searches the web, pulls **internal context with Microsoft Work IQ (CLI)**, synthesizes a competitive brief with evidence discipline, and generates a **professional slide deck** using the ppt-creator skill.

This lab pairs with Module 2 (Section 2): **automation vs. agency**, tool extensions (CLI), **skills** (building and using), and multi-agent orchestration with constructive opposition.

## What you'll build

A "Competitive Intelligence Collector" custom agent that:

1. **Searches the web** using a skill you create yourself.
1. Extracts pricing/features/claims and writes concise **summaries** to disk.
1. Pulls internal context via **Work IQ CLI**.
1. Synthesizes a competitive brief and comparison table.
1. Runs a skeptic/reviewer pass to remove unsupported claims.
1. Produces a **slide deck** using the community ppt-creator skill.
